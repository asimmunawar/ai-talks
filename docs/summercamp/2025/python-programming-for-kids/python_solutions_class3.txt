2. Count by Twos (Modified to Print Odd Numbers)
------------------------------------------------
num = int(input("Enter a number: "))
for i in range(1, num+1, 2):
    print(i)

3. Triangle Builder
-------------------
rows = int(input("How many rows? "))
for i in range(rows):
    print("X" * (i + 1))

# Optional: Balanced triangle
for i in range(rows):
    spaces = rows - i - 1
    print(" " * spaces + "X" * (2 * i + 1))
